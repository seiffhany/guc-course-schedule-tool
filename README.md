# GUC Course/Schedule tool

This tool is used to obtain your course and schedule data.

## Installation

* First

   ```commandline
   $ git clone https://github.com/seiffhany/guc-course-schedule-tool
   ```
   
* Second

   ```commandline
   $ pip install -r requirements.txt
   ```
   
* Third

   ```commandline
   $ python main.py
   ```
